# Netflix-Movies-TV-Shows-Dashboard
![Dashboard Preview](https://github.com/anasanwar19/Netflix-Movies-TV-Shows-Dashboard/blob/main/Netflix%20Moveis%20and%20TV%20Shows%20Dashboard.png)
Interactive Netflix Movies &amp; TV Shows Dashboard built using Power BI. Analyzes 8,800+ Netflix titles with insights on content type, genres, ratings, release trends, countries, and distribution through dynamic visualizations.

##  Dashboard Features
- Total Movies and TV Shows Count
- Content Distribution by Country
- Genre-wise Analysis
- Rating-wise Breakdown
- Release Year Trends
- Interactive Filters and Slicers
- Dynamic Visualizations and KPIs

##  Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Cleaning & Transformation
- Data Visualization

##  Dataset Information
The dataset contains information about Netflix titles, including:
- Title
- Type (Movie/TV Show)
- Director
- Cast
- Country
- Rating
- Release Year
- Duration
- Genre

##  Project Objective
To analyze Netflix's content catalog and create an interactive dashboard that helps users understand content trends, distribution, and viewing patterns.

##  Key Insights
- Distribution of Movies vs TV Shows
- Top Content Producing Countries
- Most Popular Genres
- Content Growth Over Time
- Rating Distribution Analysis
